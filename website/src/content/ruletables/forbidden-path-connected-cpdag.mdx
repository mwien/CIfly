Rule table used for computing the [parent adjustment distance](/applications/parent-aid/) for two CPDAGs according to the methods by <Reference id="henckelAID" format="author(year)"/>. For given sets $X$ and $Z$, it finds certain nodes $y$ such that $Z$ cannot be used as an adjustment set with respect to $X$ and $y$.
